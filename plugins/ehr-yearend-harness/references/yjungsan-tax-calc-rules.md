# yjungsan 세액 계산 규칙 사전

연말정산·원천세·지방세(사업소세 등) 계산 시 **용어 혼동으로 인한 오계산을 방지**하기 위한 참조 문서.
Claude 가 세액 계산을 수행하거나 검증할 때 반드시 이 문서를 참조한다.

> 본 문서는 코드 구현 여부와 무관한 **세법 기준 계산 규칙** 사전이다.
> 코드에 구현되어 있는지 확인하려면 `yearend-chain-tracer` 또는 `Grep` 으로 검색한다.

## 1. 핵심 용어 정의 — 이중 차감 방지

### 절대 규칙

> **과세급여(TAX_MON)는 이미 과세제외급여를 차감한 금액이다. 산출과표 산정 시 다시 빼지 않는다.**

| 용어 | 의미 | 관계 |
|------|------|------|
| **급여총액** (총지급액) | 비과세 포함 전체 급여 | = 과세급여 + 과세제외급여 |
| **과세제외급여** (비과세급여) | 지방세법 시행령 §78조의2 에 따른 비과세 급여 | 급여총액의 부분집합 |
| **과세급여** (`TAX_MON`) | 과세 대상 급여 | = 급여총액 − 과세제외급여 |
| **산출과표** | 세액 계산의 기준이 되는 금액 | **= 과세급여** (추가 차감 없음) |

### 오계산 패턴 (금지)

```
❌ 산출과표 = 과세급여 − 과세제외급여     ← 이중 차감. 절대 금지.
✅ 산출과표 = 과세급여                    ← 올바른 계산.
✅ 산출과표 = 급여총액 − 과세제외급여      ← 급여총액이 입력인 경우 올바름.
```

### 입력값 판별 기준

사용자가 금액을 제시할 때 어떤 용어를 쓰는지에 따라 산출과표가 달라진다.

| 사용자가 제시한 값 | 산출과표 |
|------------------|---------|
| "과세급여 = X" | **X** (이미 비과세 제외됨) |
| "급여총액 = X, 비과세 = Y" | **X − Y** |
| "총지급액 = X, 과세제외 = Y" | **X − Y** |

**애매할 때는 사용자에게 확인한다.** 추측으로 계산하지 않는다.

## 2. 사업소세(종업원분) 계산 규칙

### 2.1 산출세액

```
산출세액 = 산출과표 × 0.5%
```

- 10원 미만 절사

### 2.2 가산세 체계

#### 무신고가산세 (신고를 아예 안 한 경우)

```
무신고가산세 = 산출세액 × 20% × (100% − 감면비율)
```

감면비율 (2020.01.01 이후 신고분):

| 경과 기간 | 감면비율 |
|----------|---------|
| 1개월 이내 | 50% |
| 1~3개월 이내 | 30% |
| 3~6개월 이내 | 20% |
| 6개월 초과 | 0% (감면 없음) |

부정무신고가산세: 산출세액 × 40%

#### 과소신고가산세 (신고했으나 과소 신고한 경우)

```
과소신고가산세 = 산출세액 × 10% × (100% − 감면비율)
```

감면비율 (2020.01.01 이후 신고분):

| 경과 기간 | 감면비율 |
|----------|---------|
| 1개월 이내 | 90% |
| 1~3개월 이내 | 75% |
| 3~6개월 이내 | 50% |
| 6개월~1년 이내 | 30% |
| 1년~1년6개월 이내 | 20% |
| 1년6개월~2년 이내 | 10% |
| 2년 초과 | 0% (감면 없음) |

> **무신고가산세와 과소신고가산세는 택일 적용** (중복 부과 아님).

#### 납부지연가산세

```
납부지연가산세 = 산출세액 × 가산율 × 납부지연일수
```

가산율 (시기별):

| 지연일 기준 | 가산율 (%/일) |
|------------|-------------|
| 2019.01.01 이전 | 0.03% |
| 2019.01.01 이후 ~ 2022.06.06 | 0.025% |
| 2022.06.07 이후 | 0.022% |

납부지연일수: 당초 납기일 다음 날 ~ 현재 납기일 (달력일수)

> 지연 기간이 가산율 변경 시점을 걸치면 **구간별로 분리 계산** 후 합산한다.

### 2.3 절사 규칙

- **10원 미만 절사**: 산출세액, 각 가산세, 신고세액합계 모두 적용

### 2.4 최종 신고세액

```
가산세 합계 = (무신고가산세 또는 과소신고가산세) + 납부지연가산세
신고세액합계 = 산출세액 + 가산세 합계
```

## 3. 지방소득세(원천세) 가산세 규칙 (참고)

> 현재 코드에 구현된 로직: `earnIncomeTaxMgrDtl2Popup.jsp` 의 `calcPayAdtx()` 함수

| 시점 | 계산식 |
|------|--------|
| 2016.01.01 이후 | (지급액 × 0.03) + (지급액 × 지연일수 × 2.2/10000), 최대 지급액 × 0.1 |
| 2013.01.01 ~ 2016.01.01 | (지급액 × 0.05) + (지급액 × 지연일수 × 3/10000), 최대 지급액 × 0.1 |
| 2013.01.01 이전 | 지급액 × 0.1 |

## 4. 계산 시 체크리스트

Claude 가 세액 계산을 수행할 때 반드시 확인:

- [ ] 입력값이 "과세급여"인지 "급여총액"인지 확인했는가?
- [ ] 산출과표에서 이중 차감하지 않았는가?
- [ ] 10원 미만 절사를 적용했는가?
- [ ] 무신고/과소신고 가산세를 중복 부과하지 않았는가?
- [ ] 납부지연 기간이 가산율 변경 시점을 걸치는 경우 구간 분리했는가?
- [ ] 경과 기간에 따른 감면비율을 정확히 적용했는가?
